# tik-tak-toe-java-console-based
Console-based Tic Tac Toe game in C programming language with input validation and winner or draw detection

3x3 board with row/column indices
Input validation to prevent invalid or occupied moves
Winner and draw detection
Alternating turns between two players
Future Enhancements
Integrate a GUI using C graphics libraries
Add single-player mode with AI
Improve user experience with colored board and animations
